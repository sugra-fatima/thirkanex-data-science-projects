# 🔍 Exploratory Data Analysis (EDA) Project

## 🎯 Objective

To analyze a dataset and uncover patterns, trends, and relationships using statistical summaries and visualizations.

## 📂 Dataset

Job dataset containing multiple features such as:

* Job role
* Experience
* Education
* Salary

## 🛠️ Analysis Performed

### 🔹 Data Understanding

* Checked data types
* Identified missing values
* Generated statistical summary

### 🔹 Data Exploration

* Distribution analysis
* Relationship analysis
* Correlation analysis

## 📊 Visualizations

* Histogram → Salary distribution
* Scatter plot → Experience vs Salary
* Bar chart → Salary by job role
* Heatmap → Correlation matrix
* Boxplot → Outliers

## 📈 Key Findings

* Salary increases with experience
* Technical roles earn higher salaries
* Education level impacts salary
* Strong correlations between certain variables

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn

## 📌 Conclusion

EDA helps in understanding the dataset before applying machine learning. It provides insights that guide further analysis and modeling.
