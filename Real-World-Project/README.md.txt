# 🏬 Real-world Data Project (Retail Sales Analysis)

## 🎯 Objective

To perform end-to-end data analysis on a real-world retail dataset and generate meaningful business insights.

## 📂 Dataset

Kaggle Sales Dataset containing:

* Order Date
* Sales
* Profit
* Category
* Region
* Customer Segment

## 🛠️ Steps Performed

### 🔹 Data Cleaning

* Removed duplicates
* Handled missing values
* Standardized data formats

### 🔹 Exploratory Data Analysis

* Sales trends over time
* Profit analysis by category
* Regional performance analysis

### 🔹 Feature Engineering

* Extracted month/year from date
* Created new features for analysis

### 🔹 Predictive Modeling

* Built Linear Regression / Random Forest models
* Predicted sales trends

## 📊 Visualizations

* Sales by region
* Profit by category
* Monthly sales trend
* Top-performing products

## 📈 Key Insights

* Certain regions generate higher sales
* Seasonal trends impact revenue
* Some categories are more profitable than others

## 🛠️ Tools & Technologies

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

## 📌 Conclusion

Real-world data analysis helps businesses make data-driven decisions and improve strategies for growth.
