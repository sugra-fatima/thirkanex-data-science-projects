# 🤖 Predictive Modeling using Machine Learning

## 🎯 Objective

To build machine learning models that predict outcomes based on input features and evaluate their performance.

## 📂 Dataset

Job salary dataset containing:

* Experience
* Education level
* Skills count
* Industry
* Salary (target variable)

## ⚙️ Process

### 🔹 Data Preprocessing

* Encoded categorical variables using Label Encoding
* Handled missing values
* Split data into training and testing sets (80/20)

### 🔹 Models Used

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

### 🔹 Model Evaluation

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R2 Score

## 📊 Results

* Random Forest performed best due to its ability to handle complex data patterns
* Linear Regression worked as a baseline model
* Decision Tree showed some overfitting

## 📈 Key Insights

* Experience and skills strongly influence salary
* Education level has a moderate impact
* Ensemble models perform better than single models

## 🛠️ Tools & Technologies

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib

## 📌 Conclusion

Machine learning models can effectively predict outcomes and help in decision-making processes. Model selection plays a crucial role in performance.
